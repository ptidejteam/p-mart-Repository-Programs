Software::Printing

Printing

The current software adds a menu options to JBuilder 3.5 or JBuilder 3.0 Foundation and Elixir 2.4 to allow the user to print files. The program can print text files such as source code or html files. The following features are currently supported:

  • Line numbering - all files have the line number along the left side
  • Document name, page number, and time of printing - the document in a header region lists these features
  • Printing of UML class diagrams
  • Syntax highlighting for property and setting files. # is interpreted at the beginning of the line as a comment.
  • Syntax highlighting for XML and HTML files.
