Software::JRefactory - Printing and Images

There are two ways to generate output from the jrefactory tool. First you can create a JPG image of the class diagrams. Second, if the system you are using supports java 1.2 printing, you can print a picture of the software.

How to generate these output varies, but look for the print menu option for printing and the "JPG" menu option to generate the image file.
