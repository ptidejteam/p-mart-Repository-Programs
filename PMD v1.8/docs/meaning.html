What does it mean?

Here on the PMD project, we've been trying to find the meaning of the letters PMD. Because frankly, we don't really know. (We just think the letters sound good together.)

However, in the spirit of the Computing Industry, we have come up with several "backronyms" to explain it.

PMD
Pretty Much Done
Project Mess Detector
Project Monitoring Directives
Protein Mutant Database
Project Meets Deadline
Programming Mistake Detector
Pounds Mistakes Dead
